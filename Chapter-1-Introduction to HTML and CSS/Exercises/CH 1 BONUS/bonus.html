HTML & CSS REFERENCE GUIDE!

I used multiple different HTML and CSS tags and selectors to create the recipe page for the mandatory chapter 1 exercise. I will list a few, the basics at least, and simply explain them

HTML TAGS & ATTRIBUTES

  • h1 tag: main header tag
  • h2 tag: sub heading tag. theres also h3, h4, and so on but i haven't used those.
  • hr tag: visual section/page break
  • p tag: paragraph tag used for paragraphs, text, and so on.
  • img tag: allows you to display an image. make sure you put the image in src="image.png". alt="image is not working" in the img tag allows you to put a text just in case the image does not show.
  • ul tag: its used to create a bullet point list. and li tag is used to list the items
  • code tag: it displays the text given in a font used for coding to help programmers differentiate whats a code and whats normal text. example is print('hi')
  • class attribute: this is used to define or group multiple elements together. its used mainly for Javascript and CSS
  • id attribute: it's also used to define elements but it's for a singular element
  • pre tag: I didn't use this but this is a basic tag and needs to be mentioned. It has a similar use to the
            paragraph tag but it preserves line breaks and white space instead of taking up it's whole width. this text 
        is an example.

CSS DECLARATIONS AND SELECTORS

  • # selector: used to target Ids
  • . selectpr: used to select classes
  • line-height: you can change the spacing between lines in the bullet point list
  • text-align: you can align the text. examples: right, left, center
  • font-famiy: customizes font of the text
  • color: color of the text can be changed with this either using already avaible colors names like 'white' or you can change it with rgb, hsl, hex, rgba, etc.
  • text-shadow: adds stroke or shadow to the text
  • background-color: you can change the background color
